Visibility

Does the systtem keep the users informed about whats going on, through appropriate feedback?

Navigation is pretty explicit on this website. On the main page, it is easily inferred that the user can navigate to four main sections: “Study with Us”, “Our Research”, “Business & Enterprise”, “Alumni & Supporters”.

However, if the user wants to go to “Our Research”, they have to press the “Our Research” tab, and press “Our Research” again. This double click is unnecessary.

It is fairly easy to see where we are by looking at the directory-like map. This allows the user to easily navigate back to the main page. However, the aesthetics of the navigation bar could greatly be improved.
