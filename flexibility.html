Flexibility

A website should accommodate novice and experienced users, and handle direct linking well.

Overall, this site seems to have very few features for frequent users. Based on the links displayed on the homepage (and throughout the site) it appears tailored towards prospective undergraduate and graduate students, not current students or faculty. As such, the amount of “accelerators” for expert users are minimal. There are is no quick links or frequent links dropdown or menu. The only substitute for this is the links displayed in the footer, but the user must scroll all the way down the page to access this.

As far as the site’s ability to be linked to, it does not do much better. There are several pages that I was able to identify quickly that did not have any return links to the rest of the website. Two examples of this are the Prospectus form and the site search powered by Google. If a user was directly linked to either of those pages there is no easy way to return to the homepage of the site.

Excluding the pages that lack links back to the site, most of the other pages do fairly well. Most of the content pages have breadcrumbs which can take the user back to the current category’s level two page and the university's logo always redirects the user to the homepage. Furthermore, there are usually “related links” in the left and right columns which can help the user navigate to relevant pages from the page the were linked to directly.
